## 摘要
工具增强的大语言模型（LLM）智能体有望将科学推理与计算统一起来，然而，它们在药物研发等高风险领域的部署受到两个关键障碍的制约：不受约束的工具使用治理和较差的长程可靠性。在高度依赖流程的制药管线中，自主智能体往往会偏离到不可重复的轨迹中，早期的幻觉会成倍地累积成下游的失败。为克服这一问题，我们提出了 Mozi，这是一种双层架构，将生成式人工智能的灵活性与计算生物学的确定性严谨性相结合。A 层（控制平面）建立了一个受治理的“主管-执行者”层级结构，强制执行基于角色的工具隔离，将执行限制在受约束的动作空间内，并驱动基于反思的重新规划。B 层（工作流平面）将典型的药物研发阶段（从靶点识别到先导化合物优化）操作化为有状态的、可组合的技能图谱。该层集成了严格的数据契约和战略性的人机回环（HITL）检查点，以在高度不确定的决策边界保障科学有效性。秉持“安全任务自由推理，长程管线结构化执行”的设计原则，Mozi 提供了内置的鲁棒性机制和追踪级可审计性，以彻底减轻误差累积。我们在 PharmaBench（一个为生物医学智能体精心设计的基准测试）上评估了 Mozi，证明了其在编排准确性上优于现有基准。此外，通过端到端的治疗案例研究，我们展示了 Mozi 导航海量化学空间、执行严格毒性筛选以及生成极具竞争力的计算机模拟（in silico）候选药物的能力，有效地将 LLM 从脆弱的对话者转变为可靠、受控的协同科学家。

## Abstract
Tool-augmented large language model (LLM) agents promise to unify scientific reasoning with computation, yet their deployment in high-stakes domains like drug discovery is bottlenecked by two critical barriers: unconstrained tool-use governance and poor long-horizon reliability. In dependency-heavy pharmaceutical pipelines, autonomous agents often drift into irreproducible trajectories, where early-stage hallucinations multiplicatively compound into downstream failures. To overcome this, we present Mozi, a dual-layer architecture that bridges the flexibility of generative AI with the deterministic rigor of computational biology. Layer A (Control Plane) establishes a governed supervisor--worker hierarchy that enforces role-based tool isolation, limits execution to constrained action spaces, and drives reflection-based replanning. Layer B (Workflow Plane) operationalizes canonical drug discovery stages -- from Target Identification to Lead Optimization -- as stateful, composable skill graphs. This layer integrates strict data contracts and strategic human-in-the-loop (HITL) checkpoints to safeguard scientific validity at high-uncertainty decision boundaries.   Operating on the design principle of ``free-form reasoning for safe tasks, structured execution for long-horizon pipelines,'' Mozi provides built-in robustness mechanisms and trace-level audibility to completely mitigate error accumulation. We evaluate Mozi on PharmaBench, a curated benchmark for biomedical agents, demonstrating superior orchestration accuracy over existing baselines. Furthermore, through end-to-end therapeutic case studies, we demonstrate Mozi's ability to navigate massive chemical spaces, enforce stringent toxicity filters, and generate highly competitive in silico candidates, effectively transforming the LLM from a fragile conversationalist into a reliable, governed co-scientist.

### 1. 核心问题：为什么 LLM 智能体在制药领域“立不住”？
在药物研发这种高门槛领域，目前的 AI 智能体（Agent）面临两大死穴：
*   **工具调用失控（Governance）：** LLM 经常胡乱尝试高成本工具，或者在调用生物计算工具时参数格式写错（比如 PDB 蛋白质结构文件的链 ID 错误），导致程序崩溃。
*   **长程任务的误差累积（Reliability）：** 药物研发流程极长（靶点识别 -> 筛选 -> 优化）。如果第一步找错了蛋白质异构体，后面几千小时的 GPU 模拟全是无用功。现有的 Agent 缺乏“纠偏”和“硬约束”机制。

### 2. 白话版概述
Mozi（墨子）给 LLM 智能体装上了“紧箍咒”和“标准作业程序（SOP）”。它不让 LLM 漫无目的地“盲目尝试”，而是把研发流程拆成两层：上层像“主管”负责规划和反思，下层像“熟练工”按预设的图谱（Skill Graphs）执行任务。它还引入了“人机协作”关卡，在关键决策点必须由人类科学家点下“确认键”，确保 AI 不会在错误的道路上狂奔。

### 3. 方法部分：双层架构与受控自主
Mozi 的核心设计理念是：**“简单任务自由推理，长程管线结构化执行”**。

*   **A 层：控制平面（Control Plane）—— 治理引擎**
    *   **层级管理：** 采用“主管-执行者”架构。主管负责拆解任务，执行者（研究员、计算员）各司其职。
    *   **角色隔离：** 严格限制权限。比如“搜索员”不能调用昂贵的“分子对接”集群，防止资源浪费。
    *   **反思重规划：** 每一跳执行后都会自我检查，如果失败则触发重规划，而不是死循环。
*   **B 层：工作流平面（Workflow Plane）—— 技能图谱**
    *   **有状态技能图（Skill Graphs）：** 使用 LangGraph 将药物研发的四个阶段（靶点识别、活性化合物筛选、先导化合物发现、先导化合物优化）固化为有向无环图（DAG）。
    *   **数据契约（Data Contracts）：** 在工具输入输出间加装“适配器”，自动修复格式错误（如 PDBFixer 修复蛋白质侧链），确保数据流转的严谨性。
    *   **人机回环（HITL）：** 在高不确定性边界（如最终选定哪个靶点）强制暂停，等待专家审核。
*   **底层支撑：MCP 协议**
    *   通过 Model Context Protocol 统一接入了 150 多个生物数据库（UniProt, PDB 等）和计算工具（AutoDock Vina, RDKit 等）。

### 4. 实验部分：PharmaBench 基准测试
*   **数据与任务：** 提出了 **PharmaBench**，包含 88 个任务。来源包括 TDC（治疗学数据联盟）、HLE（人类最后考试，极难的专家级问题）以及生物医药数据库。
*   **Baseline：** 对比了 Biomni、STELLA、SciMaster、MiroThinker 等主流生物医药 Agent。
*   **评价指标：** 分类任务看准确率/F1，回归任务看 SMAPE（对称平均绝对百分比误差），长程任务看端到端成功率。
*   **主要结果：** 
    *   Mozi 在 PharmaBench 上的编排准确率显著高于 Biomni。
    *   在 HLE 专家级任务中，Mozi (基于 DeepSeek-V3.2) 达到了 21.4% 的准确率，优于 Gemini-2.5-Pro。
    *   **案例研究：** 在克罗恩病、帕金森病、败血症三个案例中，Mozi 成功从零开始识别靶点并生成了具有竞争力的候选分子。
